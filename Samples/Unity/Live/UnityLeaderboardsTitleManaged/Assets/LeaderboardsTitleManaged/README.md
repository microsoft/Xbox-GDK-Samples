# Microsoft GDK Sample: Leaderboards (Title-Managed)

**Relevant areas:** User sign-in, Xbox Live Leaderboards with Title-Managed stats

## Description

This sample demonstrates basic usage of Xbox Live and Leaderboards to perform the following:

- Login to Xbox Live services

- Retrieve Xbox Live profile information

- Query global and social leaderboards

- Query and update user statistics

For more information about Unity samples, see the [Microsoft GDK API package > Samples](https://docs.unity3d.com/Packages/com.unity.microsoft.gdk@1.3/manual/Samples.html) documentation.
