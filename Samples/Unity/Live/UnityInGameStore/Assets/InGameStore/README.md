# Microsoft GDK Sample: In Game Store

**Relevant areas:** User sign-in, XStore commerce and licensing

## Description

This sample demonstrates in-depth usage of Xbox Live and XStore APIs to perform the following:

- Login to Xbox Live services

- Retrieve Xbox Live profile information

- Retrieve Microsoft Store catalog details for product display and purchase within the title

- Retrieve user entitlements to determine product ownership/access

- Acquire a game license and check for type/validity

- Preview/acquire durable licenses and listen for license lost events

- Support for game trials and upgrades

For more information about Unity samples, see the [Microsoft GDK API package > Samples](https://docs.unity3d.com/Packages/com.unity.microsoft.gdk@1.3/manual/Samples.html) documentation.
