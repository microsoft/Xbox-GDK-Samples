# Microsoft GDK Sample: Simple MPSD

**Relevant areas:** User Sign-in, Multiplayer Session Directory (MPSD), SmartMatch

## Description

This sample demonstrates basic usage of Xbox Live, MPSD, and SmartMatch APIs to perform the following:

- Login to Xbox Live services

- Retrieve Xbox Live profile information

- Start an Xbox Live multiplayer session using MPSD

- Invite Xbox Live friends to join your multiplayer session

- Join a friend's Xbox Live multiplayer session

- Start and cancel a SmartMatch multiplayer session

- Leave a multiplayer session

This sample does not cover the full breadth of MPSD's features.
Please refer to the [MPSD GDK documentation](https://learn.microsoft.com/en-us/gaming/gdk/_content/gc/live/features/multiplayer/mpsd/live-mpsd-overview) for detailed API notes and usage.

For more information about Unity samples, see the [Microsoft GDK API package > Samples](https://docs.unity3d.com/Packages/com.unity.microsoft.gdk@1.3/manual/Samples.html) documentation.
