# Microsoft GDK Sample: Simple Web Sockets

**Relevant areas:** User sign-in, Web Sockets

## Description

This sample demonstrates usage of Xbox Live and LibHttpClient to perform the following:

- Login to Xbox Live services

- Retrieve Xbox Live profile information

- Use LibHttpClient to connect, send, and receive messages via Web Sockets

For more information, refer to the 'Samples' section of the com.unity.microsoft.gdk package documentation.
