# Microsoft GDK Sample: Achievements (Title-Managed)

**Relevant areas:** User sign-in, Title-Managed Achievements

## Description

This sample demonstrates basic usage of Xbox Live and Title-Managed Achievements to perform the following:

- Login to Xbox Live services

- Retrieve Xbox Live profile information

- Query Xbox Live for all achievements for the title

- Querying for status on a single achievement

- Updating the progress of an achievement

For more information about Unity samples, see the [Microsoft GDK API package > Samples](https://docs.unity3d.com/Packages/com.unity.microsoft.gdk@1.3/manual/Samples.html) documentation.
