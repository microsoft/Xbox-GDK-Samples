# Microsoft GDK Sample: Simple Http

**Relevant areas:** User sign-in, HTTP requests with XSAPI

## Description

This sample demonstrates usage of Xbox Live and Xbox Services API (XSAPI) to perform the following:

- Login to Xbox Live services

- Retrieve Xbox Live profile information

- Create an HTTP handle to track the REST request

- Send an HTTP request to an Xbox Live service as string or byte data

- Retrieve the HTTP response

- Close the HTTP handle

For more information, refer to the 'Samples' section of the com.unity.microsoft.gdk package documentation.
