# Microsoft GDK Sample: Simple PLM

**Relevant areas:** User sign-in, process lifetime management (PLM), XGameShowUi, XLaunchUri

## Description

This sample demonstrates how to use Xbox Live, XGameShowUi, and XLaunchUri to handle PLM events.
The sample provides the following behaviors:

- Login to Xbox Live services

- Retrieve Xbox Live profile information

- Call XGameShowUi* to display TCUI and trigger focus lost (PC) or constrain (Xbox console) events

- Call XLaunchUri to launch into Settings and trigger focus lost (PC) or constain (Xbox console) events

- Handle suspend events from the system (Xbox console)

For more information about Unity samples, see the [Microsoft GDK API package > Samples](https://docs.unity3d.com/Packages/com.unity.microsoft.gdk@1.3/manual/Samples.html) documentation.
